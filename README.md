# react-native-quick-buffer

Node `buffer` for React Native built with JSI

## Installation

```sh
npm install react-native-quick-buffer
```

## Usage


```js
import { multiply } from 'react-native-quick-buffer';

// ...

const result = multiply(3, 7);
```
